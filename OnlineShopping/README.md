OnlineShopping - Minimal JSP/Servlet/JDBC Dynamic Web Project
------------------------------------------------------------
How to use:
1. Unzip this project and import to Eclipse as Existing Dynamic Web Project (or copy into your workspace).
2. Add MySQL Connector/J jar to WebContent/WEB-INF/lib (or project's build path).
3. Create MySQL DB and run sql/schema.sql. Update DB credentials in DBConnection.java.
4. Run on Tomcat server from Eclipse.
